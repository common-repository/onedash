[OneDash](https://onedash.com/) plugin is solution to be able to use OneDash without any external plugin or software.

== Description ==

[OneDash](https://onedash.com/) is a solution for monetizing video-based media. Our goal is giving media owners, advertisers,
and creators the tools to push content over any platform to maximize value. The primary product of OneDash
is our content dashboard comprised of various solutions and multi-faceted tools that, when put together,
offer a turnkey approach to the value creation of video-based media.

= Plugin Features =

*   Bootstraps OneDash without installation any external plugins.

== Installation ==

1.  Upload plugin folder `onedash` to the '/wp-content/plugins' directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
